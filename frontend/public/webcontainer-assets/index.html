Getting Started

To see your application, you need to:

  1. Mount your files to the WebContainer
  2. Run your application using the terminal
  3. Access your application at the appropriate port

Example Commands

# List files
ls -la

# Install dependencies
npm install

# Start a Node.js application
node index.js

# Start a development server
npm run dev

Use the terminal in the editor to run commands in this environment.

Your files have been successfully mounted to the WebContainer.
